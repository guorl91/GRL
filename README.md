# 时代少年团
## 基本信息
时代少年团（Teens in Times、TNT）是由时代峰峻推出的中国内地男子演唱组合，由**马嘉祺**、**丁程鑫**、**宋亚轩**、**刘耀文**、**张真源**、**严浩翔**、**贺峻霖**七人组成。
### 成立时间
2019年
### 经纪公司
*北京时代峰峻文化艺术发展有限公司*
### 百度百科
